<p>The ?matches built-in will allow you to do regular-expression searches on a given string. It returns a list of substring matches too, so you should be able to get the SRC attribute, put it into a variable, and then build up your PHP code around that.</p>

<p>See the Freemarker manual for details on how to use the built-in:</p>

<p><a href="http://freemarker.sourceforge.net/docs/ref_builtins_string.html#ref_builtin_matches" rel="nofollow">http://freemarker.sourceforge.net/docs/ref_builtins_string.html#ref_builtin_matches</a></p>
