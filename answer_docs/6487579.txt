<p><code>Double.valueOf(string)</code> returns a <code>Double</code><br>
<code>Double.parseDouble(string)</code> returns the primitive type.<br>
Since you are assigning the result to a <code>Double</code>, it might be better to use<br>
<code>Double.valueOf(string)</code>.<br>
I can't get the code to break either way, but I'd say it's worth a shot.</p>
