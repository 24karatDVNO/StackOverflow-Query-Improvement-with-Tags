<p>You can change the keyboard available by using the inputType for the <code>EditText</code>.</p>

<pre><code>&lt;EditText android:inputType="number"/&gt;
</code></pre>

<p><a href="http://developer.android.com/reference/android/widget/TextView.html#attr_android%3ainputType" rel="nofollow">XML documentation</a></p>

<pre><code>editText.setInputType(int);
</code></pre>
