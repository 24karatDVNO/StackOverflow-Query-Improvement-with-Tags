<p>Take a look at the <a href="http://javamail-crypto.sourceforge.net/" rel="nofollow">JavaMail Crypto</a> and <a href="http://bouncycastle.org" rel="nofollow">Bouncy Castle</a> combo.</p>

<p>They allow you to import the p12 file using a file input stream and let you deal with it accordingly.</p>
