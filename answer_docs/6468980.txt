<p>Flash is quite old stuff for building (simple) animations, think of it like a way to create graphic elements and animations.</p>

<p>Flex is a framework for the developers. You can develop applications with it.</p>

<p>A software developer should definitely go for Flex, on the other hand there might be some clicking tools for building Flash animations for people who do not wish to program.</p>

<p>You can hardly distinguish how the sites were built, because the result gets compiled into binary file (you might do a guess based on the file size in some cases)</p>
