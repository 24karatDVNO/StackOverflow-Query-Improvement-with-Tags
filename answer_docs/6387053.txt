<p>When you run <code>val++</code>, you just increment the value stored in the variable.</p>

<p>When you run <code>val=val+1</code>, <code>1</code> is an <code>int</code> by default so the compiler considers both values to be <code>ints</code> and returns an <code>int</code> with the value <code>val+1</code>. Then it fails to put it back into <code>val</code> because an <code>int</code> is to large to store in a short.</p>
