<p>If you look in the <a href="http://developer.apple.com/library/ios/#DOCUMENTATION/UIKit/Reference/UIDatePicker_Class/Reference/UIDatePicker.html" rel="nofollow"><code>documentation</code></a> you will see that there is a property of <code>UIDatePicker</code> called <code>maximumDate</code>. If you then look at the documentation for <a href="http://developer.apple.com/library/ios/#DOCUMENTATION/Cocoa/Reference/Foundation/Classes/NSDate_Class/Reference/Reference.html#//apple_ref/doc/c_ref/NSDate" rel="nofollow"><code>NSDate</code></a> you will see that the class method <code>date</code> returns an NSDate with the current date and time, therefore:
`someDatePicker.maximumDate = [NSDate date];
will set the date picker to not allow any date (or time) later than that moment.</p>
