<p>It's not assembly language, but MSIL.  You can use MSIL Disassembler to view MSIL from an assembly (.dll or .exe) : <a href="http://msdn.microsoft.com/en-us/library/f7dy01k1(v=vs.80).aspx" rel="nofollow noreferrer">http://msdn.microsoft.com/en-us/library/f7dy01k1(v=vs.80).aspx</a></p>

<p>Even better than that is a tool like Reflector: <a href="http://reflector.red-gate.com/download.aspx?TreatAsUpdate=1" rel="nofollow noreferrer">http://reflector.red-gate.com/download.aspx?TreatAsUpdate=1</a> - a bit more interactive.</p>
