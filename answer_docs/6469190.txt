<p>There are some 3D intrinsics (called 3DNow) which are intended to be used mainly by game programmers. Some of this functions may be used for other type of computation (like algebra or fibo), but don't expect to be easy.</p>

<p>You can take a look in header file "mm3dnow.h" to make an idea. or on <a href="http://msdn.microsoft.com/en-us/library/ccky3awe%28v=VS.80%29.aspx" rel="nofollow">msdn</a></p>
