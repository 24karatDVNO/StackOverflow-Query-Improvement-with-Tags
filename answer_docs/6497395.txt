<p>You have to follow the new naming scheme (idiom in Apple-speak) specified for <a href="http://developer.apple.com/library/ios/#documentation/iPhone/Conceptual/iPhoneOSProgrammingGuide/BuildTimeConfiguration/BuildTimeConfiguration.html#//apple_ref/doc/uid/TP40007072-CH7-SW24" rel="nofollow">Universal Applications</a> for the <code>NSMainNibFile</code> key in your info.plist. For example, if you set your NSMainNibFile to be "MainWindow-iPhone", the xib for ipad would be "MainWindow-iPad". </p>

<p>The same naming convention should hold for the views in a view based application (I can't test right now, installing new xcode).</p>
