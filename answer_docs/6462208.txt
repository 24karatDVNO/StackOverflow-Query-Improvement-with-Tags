<p>PHP populates <code>$_POST</code> by parsing the data received. However, it only knows form-encoded data, JSON data cannot be parsed automatically. So <code>$_POST</code> will be useless in this case. You need to <a href="http://php.net/%24HTTP_RAW_POST_DATA" rel="nofollow">get the raw post data</a> and parse it with <code>json_decode</code>.</p>
