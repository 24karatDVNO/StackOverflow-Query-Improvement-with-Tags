<p>Logically (and generally) all such 'meta' data in the database are treated structurally the same way as user data; the system simply maintains one or more schemas dedicated to system level data.  </p>

<p>Physical organization of databases are very much vendor specific e.g. an Oracle 10g database will organize the data on disk in a manner specific to it.  For this case, you are more likely to get concrete feedback if you note which database product you are using.</p>
