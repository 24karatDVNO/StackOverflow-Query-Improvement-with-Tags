<p>Which calendar do you use? If it does not matter, I recommend <a href="http://jqueryui.com" rel="nofollow">jQuery UI Calendar</a>, it's flexible, few clicks themeable &amp; easy to learn. You can add it with just few line of code, for example </p>

<pre><code>&lt;script&gt;
    $(function() {
        $( "#datepicker" ).datepicker();
    });
&lt;/script&gt;
</code></pre>

<p>And current date is selected by default. See <a href="http://jqueryui.com/demos/datepicker/" rel="nofollow">this</a> page for demo and code samples.</p>
