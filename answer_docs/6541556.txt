<p><code>simple_xml_load_file('yourfile.php')</code> will do a file-based open/read, and you'll be loading raw PHP code directly into simplexml. <code>simple_xml_load_file('http://example.com/yourfile.php')</code> will do a full-blown HTTP request and cause yourfile.php to be executed, at which point it should output the xml header and output xml text.</p>
