<p>ListBox has <a href="http://msdn.microsoft.com/en-us/library/system.windows.forms.listbox.items.aspx" rel="nofollow">Items</a> property. You can use <a href="http://msdn.microsoft.com/en-us/library/system.windows.forms.listbox.objectcollection.add.aspx" rel="nofollow">Add()</a> method to add object to list.</p>

<pre><code>listBox.Items.Add("My new list item");
