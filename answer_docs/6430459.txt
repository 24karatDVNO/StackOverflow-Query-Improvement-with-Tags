<p>In XML DTDs, there is no elegant way to specify this constraint. You can do as @DevNull suggests; it is legal, but the technique becomes unwieldy when the number of elements is large.</p>

<p>If this had been about an SGML DTD, then you could have used the <a href="http://www.xmlgrrl.com/publications/DSDTD/apa.html#elemdecls" rel="nofollow"><code>&amp;</code> (AND) connector</a>. But that is one of the features that was <a href="http://www.w3.org/TR/NOTE-sgml-xml.html" rel="nofollow">removed</a> when XML was created.</p>
