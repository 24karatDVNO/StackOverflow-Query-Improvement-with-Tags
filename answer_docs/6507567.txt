<p>There is no system support for extending the standard progress dialog. However you could find the HWND of the dialog, and then add your own controls.</p>

<p>Perhaps it's better if you create your own <a href="http://msdn.microsoft.com/en-us/library/hh298398%28v=vs.85%29.aspx" rel="nofollow" title="How to Create an Animation Control">animation control</a>.</p>
