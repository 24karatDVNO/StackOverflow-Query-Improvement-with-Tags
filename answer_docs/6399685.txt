<p>Just call the Facebook graph api by accessing the app's name or id:
<a href="http://graph.facebook.com/appId" rel="nofollow">http://graph.facebook.com/appId</a>
or <a href="http://graph.facebook.com/appName" rel="nofollow">http://graph.facebook.com/appName</a></p>

<p>If you are trying to search for apps, you could scrape this url:
<a href="https://www.facebook.com/search.php?q=appNameSearch&amp;type=apps" rel="nofollow">https://www.facebook.com/search.php?q=appNameSearch&amp;type=apps</a></p>
