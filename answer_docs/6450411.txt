<p>you can create your own <code>drawable</code> folder in <code>res</code> directory. But remember keep the images in that folder which are common for all screen size devices. <code>drwable-hdpi</code> means this directory contained the images will be loaded when the device has higher dpi. similarly <code>drawable-mdpi</code> and <code>drawable-ldpi</code> are there.</p>
