<p>In Qt a widgets size is usually controlled by its <a href="http://doc.trolltech.com/4.7/qlayout.html" rel="nofollow">layout</a>. The <a href="http://doc.trolltech.com/4.7/layout.html" rel="nofollow">layout management</a> docs should help. In particular, take a look at <a href="http://doc.trolltech.com/4.7/qsizepolicy.html" rel="nofollow">QSizePolicy</a>. If you're using designer, you can set these directly as properties, otherwise you'll need to do it in code. The <a href="http://doc.trolltech.com/4.7/layouts-basiclayouts.html" rel="nofollow">basic layouts</a> example may help as well.</p>
