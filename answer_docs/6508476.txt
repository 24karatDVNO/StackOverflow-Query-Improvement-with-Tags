<p>Given these two premises:</p>

<ul>
<li>you're using Visual Studio 2010 Premium or Ultimate</li>
<li>your unit tests are written using MSTest</li>
</ul>

<p>then you could use the <a href="http://msdn.microsoft.com/en-us/library/dd264992.aspx" rel="nofollow">Test Impact View</a> to look at and run the unit tests that are affected by the code changes made since the last time the solution was built on your local machine.</p>
