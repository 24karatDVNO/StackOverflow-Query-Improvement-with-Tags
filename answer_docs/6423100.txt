<p><code>int main() {}</code> this is the name of the method</p>

<p><code>int x, y;</code> this is declaring two integers with no values stored in them (e.g. x=no number &amp; y = no number).</p>

<p><code>for(x = 10; x &gt;= 1; x--)</code>  this sets x = 10 then executes the code within the given block multiple times decrementing x by 1 through each pass up until it reaches 1 which then stops the loop.</p>

<p><code>for(y = 1; y &lt; x; y++)</code>  this sets y = 1 then executes the code within the given block multiple times increasing y by 1 through each pass up until it reaches a value below x.</p>

<p><code>printf("%d", y);</code>  this prints the value of y.
 <code>printf("\n");</code>    this starts a new line
 <code>system("pause");</code> this pauses the system
 <code>return 0;</code> this returns 0 thus ending the program</p>
