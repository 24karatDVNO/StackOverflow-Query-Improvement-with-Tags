<p>When you use <a href="http://doc.qt.nokia.com/latest/qwidget.html#size-prop" rel="nofollow">resize</a> or <a href="http://doc.qt.nokia.com/latest/qwidget.html#geometry-prop" rel="nofollow">setGeometry</a> methods, your rectangle is bound by <a href="http://doc.qt.nokia.com/latest/qwidget.html#minimumSize-prop" rel="nofollow">minimum</a> size of the widget.</p>

<p>By default this is 0,0 but is there any chance that you are setting minimum size in your code perhaps by the layout so that resize method rejects your given parameter ?</p>
