<p>I have written a matcher framework for this:</p>

<p>The base interface is Matcher (similar to Comparable):
<a href="http://softsmithy.sourceforge.net/lib/docs/api/org/softsmithy/lib/util/Matcher.html" rel="nofollow">http://softsmithy.sourceforge.net/lib/docs/api/org/softsmithy/lib/util/Matcher.html</a></p>

<p>MatchableObject is a class, which applies a matching strategy to an Object: 
<a href="http://softsmithy.sourceforge.net/lib/docs/api/org/softsmithy/lib/util/MatchableObject.html" rel="nofollow">http://softsmithy.sourceforge.net/lib/docs/api/org/softsmithy/lib/util/MatchableObject.html</a></p>

<p>Matchers is a utility class to wrap/ unwrap MatchableObjects:
<a href="http://softsmithy.sourceforge.net/lib/docs/api/org/softsmithy/lib/util/Matchers.html" rel="nofollow">http://softsmithy.sourceforge.net/lib/docs/api/org/softsmithy/lib/util/Matchers.html</a></p>

<p>The library is Open Source. Maybe you find it useful.</p>
