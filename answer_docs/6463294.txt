<p>Just be sure you use the same order. If the function is declared as:</p>

<pre><code>function(a,b){}
</code></pre>

<p>Then the attributes must be a,b; not b,a.</p>

<p>Alternatively, if you really want to name them, use a colon in front of the attribute name.</p>
