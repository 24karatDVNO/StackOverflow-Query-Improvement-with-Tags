<p>You will want to create a <a href="http://www.w3schools.com/html/html_forms.asp" rel="nofollow">HTML form</a> that submits a POST request back to a PHP file on your site. In that page you will extract the <a href="http://php.net/manual/en/reserved.variables.post.php" rel="nofollow">$_POST</a> variable, validate it, then use it to post a new status like <code>$connection-&gt;post('statues/update', array('status' =&gt; $newStatusTextFromPost));</code></p>
