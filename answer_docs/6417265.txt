<p>If it's a console application then it probably won't open the command prompt on double click although the app may be running in the background. I recommend making batch files for console applications.</p>

<p>If its a Swing application and still won't run on double click you should try wrapping as an exe file using this <a href="http://launch4j.sourceforge.net/" rel="nofollow">http://launch4j.sourceforge.net/</a></p>
