<p>When a user likes your Facebook fan page, Facebook will reload the whole page.  You can then use server code to inspect the signed_request post parameter that they will send over. <a href="https://stackoverflow.com/questions/5093398/check-if-user-liked-page/5100287#5100287">This question</a> has an example code of how to detect if they liked the page in server side code. </p>
