<p>Check the the minimum size you specified when you create your databases.Check your database growth settings etc.minimum size of a database is  specified when the database was originally created, or the last  size set by using a file-size-changing operation, such as DBCC SHRINKFILE. For example, if a database was originally created with a size of 4GB and grew to 4.1 GB, the smallest size the database could be reduced to is 4GB, even if all the data in the database has been deleted.</p>
