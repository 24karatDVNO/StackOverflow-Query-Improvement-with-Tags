<p>Check following parameters on your manifest file</p>

<p>android:minSdkVersion</p>

<pre><code>This is an integer that indicates the minimum API Level required to properly run your application.
</code></pre>

<p>android:targetSdkVersion</p>

<pre><code>An integer designating the API Level that the application is targetting. 
</code></pre>
