<p>How are you putting the user text?  If you have control, from the server side all    languages support encoding the output prior to rendering it on the browser.  </p>

<p>As long as you sanitize the output.  You can basically allow the user to enter anything that they wish.  Then when displaying the content, you just need to encode it so that the browser will not execute it.  This can be done from any language PHP, JScript, ASP, VB, C#, etc...</p>
