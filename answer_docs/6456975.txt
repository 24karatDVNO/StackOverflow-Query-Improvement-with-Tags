<p>You generally can't change the appearance of a file input selector, but you can use some <a href="http://www.quirksmode.org/dom/inputfile.html" rel="nofollow">CSS tricks to disguise it</a>.  Supposedly, an <code>&lt;input type="file"/&gt;</code> element should support an <code>accept</code> property whose value is a MIME type for the desired type of input file, but that <a href="http://www.w3schools.com/TAGS/att_input_accept.asp" rel="nofollow">isn't implemented</a>.</p>
