<p>I'm no Flex expert, but this is possible without too much trouble (using Flex SDK 3.2, anyway). You're right - when you turn off the <code>horizontalScrollPolicy</code>, the <code>maxHorizontalScollPosition</code> is set to 0, UNLESS you specify both a <code>width</code> value AND a <code>maxWidth</code> value. Then, <code>maxHorizontalScrollPosition</code> will again contain a useable value, and you'll be able to programmatically set the <code>horizontalScrollPosition</code>.</p>
