<p>Java's <a href="http://download.oracle.com/javase/6/docs/api/java/lang/reflect/Proxy.html" rel="nofollow">Proxy</a> class can be used to create dynamic classes that implement specific interfaces at runtime. Such classes use an <a href="http://download.oracle.com/javase/6/docs/api/java/lang/reflect/InvocationHandler.html" rel="nofollow">InvocationHandler</a> to handle method calls.</p>
