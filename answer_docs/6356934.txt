<p>The answers given are probably the best approach for now but once you've been using emacs for a while you should look at <a href="http://www.emacswiki.org/emacs/InteractivelyDoThings" rel="nofollow">ido-mode</a>. Type <code>M-x ido-mode</code> to start, then use the arrow keys to move quickly around file paths.</p>
