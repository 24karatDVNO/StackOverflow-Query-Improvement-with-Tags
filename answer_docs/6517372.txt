<p>To clear this error you need to shut netbeans, go to the .netbeans folder in your home directory  C:\users\.netbeans for Win 7, 
in there you will see a folder for the version you are using.
in there is a folder called var
delete this folder then re-run netbeans.</p>

<p>This clears the cache in Netbeans allowing it to re scan the folders correctly.</p>
