<p>Take a peek in .git/config for your project - you will see a [remote="heroku"] section or there (or not) which you can delete - this is the same as doing git remote rm heroku from the command line. </p>

<p>Then you'll need to readd your new heroku git url (which you can get from the myapps page on the Heroku site) and then do git add heroku . Then it should all work when you try doing a push.</p>
