<p>The html tags required to embed an swf file are different for some browsers.</p>

<p>Firefox, Chrome, .. use <code>&lt;object&gt;</code> tags while <code>IE</code> uses  tags.</p>

<p>Try using <a href="http://code.google.com/p/swfobject/" rel="nofollow">SWFObject</a>, which will handle cross browser problems for you.<br/>
It even comes with a <a href="http://code.google.com/p/swfobject/downloads/detail?name=swfobject_generator_1_2_html.zip&amp;can=2&amp;q=" rel="nofollow">handy generator</a> to assist you with embedding.</p>
