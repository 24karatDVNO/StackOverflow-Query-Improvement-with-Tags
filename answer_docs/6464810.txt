<p>What is windows log in? There is only user log in. If you need service host application to start after computer boot you need to <a href="http://msdn.microsoft.com/en-us/library/ms733069.aspx" rel="nofollow">host it in Windows Service</a>. If you need some process to start after user log in you can create simple console application and add it to <code>Startup</code> (applications running after user log in).</p>

<p>Anyway creating WCF service for copying files is a global nonsense. It looks like everybody things that WCF is technology for large data transfers. It is not - especially it is not a technology for file transfers. Use shared drives and normal file copying - custom batch file can be added on each machine into <code>Startup</code> or install FTP (part of IIS which is available on Windows 7 Ultimate).</p>
