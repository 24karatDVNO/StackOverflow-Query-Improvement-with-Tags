<p>Printing the InetAddress objects getHostAddress() method prints a textual representation of the address (as specified in the API, check <a href="http://download.oracle.com/javase/6/docs/api/java/net/InetAddress.html#getHostAddress" rel="nofollow">http://download.oracle.com/javase/6/docs/api/java/net/InetAddress.html#getHostAddress</a>()). This should be the IP address of your computer? (Or possible 127.0.0.1, the address that always points to your local computer).</p>

<p>The Socket class (http://download.oracle.com/javase/6/docs/api/java/net/Socket.html) takes an InetAddress + port number in the constructor so you should be able to pass it the InetAddress and the first parameter and the port number that your SocketServer listens on. (Each server socket opens up a listener on a port number which you can basically choose yourself, check <a href="http://download.oracle.com/javase/6/docs/api/java/net/ServerSocket.html" rel="nofollow">http://download.oracle.com/javase/6/docs/api/java/net/ServerSocket.html</a>).</p>
