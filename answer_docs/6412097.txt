<p>Depending on how many users and objects there are, it's often just as fast or faster to drop the existing database with <a href="http://download.oracle.com/docs/cd/B19306_01/server.102/b14200/statements_8009.htm#SQLRF01513" rel="nofollow">DROP DATABASE</a> and re-create it using the database creation wizard. The wizard will allow you to save your creation steps in a template so that this process is very easy next time.</p>
