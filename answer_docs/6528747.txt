<p>You should use a different repository for each project. This is because when you do a commit in hg, the commit is for the entire hierarchy, not just the directory (and below) that you are in. For more details have a look at the <em>One last big conceptual difference</em> section of <a href="http://hginit.com/00.html" rel="nofollow">http://hginit.com/00.html</a></p>
