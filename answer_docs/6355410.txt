<p>How to set a manual IP address on your printer:</p>

<p>When the printer is connected to the network the first time, the router will assign an IP address to it. To get the IP address that the printer was assigned, print a configuration page from the printer. It will show you the IP address on that page somewhere.</p>

<p>Using an Internet browser, type the IP address of the printer into the address line of the browser and press enter. It should bring up a printer configuration window.</p>
