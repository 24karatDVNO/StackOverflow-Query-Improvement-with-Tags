<p>The snippet that the installer gives you need to go in a file called the bashrc. The file lives in your home directory: /Users/eric/.bashrc</p>

<p>You need to edit this file and add the line from rvm and then you should be good to go. </p>

<p>As for getting rolling with rails I'd recommend The Pragmatic Programmers book on rails. You can find their books at pragprog.com</p>
