<p>It sounds like the code is expecting an <code>oauth_callback_confirmed</code> parameter to be returned from Twitter when there isn't one. <code>oauth_callback_confirmed</code> is returned when you specify an <code>oauth_callback</code> URL in the request to get the access_token. You can read more about <code>oauth_callback</code> on <a href="https://dev.twitter.com/pages/auth#request-token" rel="nofollow">dev.twitter.com</a>.</p>
