<p>Are you using a custom cell? you can create the same effect by using UITableViewCellStyleValue1 (that controls itself your problem, and move automatically the label to left) and then by adding programmatically the UILabel to each cell in cellForRowAtIndexPath.
If you are using UITableViewCellStyleSubtitle try to use UITableViewCellStyleValue1 and add the UIlabel on the bottom of the cell.textLabel property.</p>
