<p>If you mean the <strong>option menu</strong>, then this guide should help you: <a href="http://developer.android.com/guide/topics/ui/menus.html#ChangingTheMenu" rel="nofollow">Changing the Menu</a>. </p>

<p>It shows how you can dynamically add and remove items from the menu right before it is shown to the user using the <a href="http://developer.android.com/reference/android/app/Activity.html#onPrepareOptionsMenu%28android.view.Menu%29" rel="nofollow">onPrepareOptionsMenu()</a> method, which passes you the menu. You can then add or remove items from the menu. </p>
