<p>You should use <a href="http://www.asp.net/security/tutorials/an-overview-of-forms-authentication-cs" rel="nofollow">Forms Authentication</a> - on click of login button on Login Form, you need to validate user supplied credentials by making the web service call. In case the credentials are correct then you can use <a href="http://msdn2.microsoft.com/en-us/library/system.web.security.formsauthentication.redirectfromloginpage.aspx" rel="nofollow">FormsAuthentication.RedirectFromLoginPage</a> to indicate authentication success to ASP.NET runtime.</p>

<p>Yet another variation (along with Forms Authentication) will be to use custom membership provider. See this <a href="http://msdn.microsoft.com/en-us/library/tw292whz.aspx" rel="nofollow">MSDN link</a> to understand how membership works. Then you have to <a href="http://msdn.microsoft.com/en-us/library/f1kyba5e.aspx" rel="nofollow">write a custom membership provider</a> that will use the web service to validate user.</p>
