<p>On each windows system you can find ressource files with icons used by the shell <code>Explorer.exe</code>.</p>

<p>From Windows NT 3.1 to 5.2 (W2K3 or XP) ressources are in :</p>

<pre><code>C:\Windows\System32\Shell32.dll
</code></pre>

<p>Beginig NT 6.0 (Vista) they are in :</p>

<pre><code>C:\Windows\System32\Imageres.dll
</code></pre>

<p>You can open these files with Visual Studio as ressources files to find the Shell Icons you need.</p>
