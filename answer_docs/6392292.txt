<p>Another option would be to use the <code>parse</code> option of <a href="http://docblox-project.org" rel="nofollow">DocBlox</a> to generate a XML structure which you can then interpret with your own code.</p>

<p>After you have installed DocBlox using <a href="http://pear.docblox-project.org" rel="nofollow">PEAR</a> you can use the following command to generate a structure.</p>

<pre><code>docblox parse -d [SOURCE_FOLDER] -t [TARGET_LOCATION]
</code></pre>

<p>or</p>

<pre><code>docblox parse -f [SOURCE_FILE] -t [TARGET_LOCATION] 
</code></pre>
