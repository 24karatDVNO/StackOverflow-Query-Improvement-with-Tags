<p>Appcelerator Titanium only supports iPhone and Android apps. Blackberry support is in progress but not finished now. (See <a href="http://www.appcelerator.com/products/titanium-mobile-application-development/" rel="nofollow">Appcelerator Website</a>) There is no Symbian support.</p>

<p>PhoneGap supports more platforms including Symbian. (See <a href="http://www.phonegap.com/start" rel="nofollow">PhoneGap Website</a>)</p>
