<p>You need to use the 1.2.5 version of the <code>sqlite3-ruby</code> gem.</p>

<pre><code>gem install sqlite3-ruby --version=1.2.5
</code></pre>

<p>You can either make sure there is no other gems providing sqlite3 (such as sqllite3 or a newer version of sqllite3-ruby) or explicitly require this version of this gem.</p>
