<p>For Aptana Studio 3: (plugin for Eclipse tested)</p>

<p>Find a library you are interested in...Save it to your disk, and then drag it into your project. It does not matter where in the project it sits, so you can create a new folder for files like this if you like.</p>

<p><strong>!! The project must be some type of web project: Web, PHP, Python, Ruby, Rails.</strong> </p>
