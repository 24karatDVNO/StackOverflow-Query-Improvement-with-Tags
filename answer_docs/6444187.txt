<p>If you are trying to create your own site that gets data from the users latest wall posts, you can use the Facebook Graph API <a href="https://developers.facebook.com/docs/reference/api/" rel="nofollow">https://developers.facebook.com/docs/reference/api/</a> </p>

<p>To pull most recent wall posts (in JSON format), use:</p>

<p><code>https://graph.facebook.com/me/feed?access_token=&lt;UserTokenHere&gt;</code></p>

<p>The Facebook Graph API site above gives lots of good info on how to use this. If you are trying to do something else, please clarify your question.</p>

<p>Cheers!</p>
