<p>Tools -> Options -> Environments holds all modifications you are allowed to make.  If it is not in this list, it can't be done with the default install.  For HTML though, you can make those types of changes for specific tags, for example, by changing the formatting for the tag.</p>

<p>Tools -> Options -> Environments -> Keywords -> Keyword -> Item foreground</p>

<p>Tools -> Options -> Environments -> Keywords -> Identifier -> Item foreground</p>

<p>You cannot however do exactly what you are wanting to do but, at the very least, you now know how to make some modifications.</p>
