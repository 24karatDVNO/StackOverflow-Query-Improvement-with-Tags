<p>Did you enable NSZombies?</p>

<p>If not here is how you do it?</p>

<ul>
<li>Goto: Product-> Edit scheme</li>
<li>Choose Ran tab  </li>
<li>Select "Arguments"  </li>
<li>In the Environment Variables add  </li>
<li>NSZombieEnabled and set its value to YES  </li>
</ul>

<p>Run your code. It should tell you where you have an error. </p>
