<p>For readability, I prefer</p>

<pre><code>if (conditiontest) doOneThing();
</code></pre>

<p>over either of these</p>

<pre><code>if (conditiontest) { doOneThing(); }

if (conditiontest) {
     doOneThing();  }

if (conditiontest) { 
     doOneThing(); 
     }
</code></pre>

<p>This mostly just my opinion.  But I think it enhances it's succinctness. </p>
