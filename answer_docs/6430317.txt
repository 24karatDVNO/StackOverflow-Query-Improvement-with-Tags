<p>The short answer: Multiple inheritance is not allowed in C#. Read up on interfaces: <a href="http://msdn.microsoft.com/en-us/library/ms173156.aspx" rel="nofollow">http://msdn.microsoft.com/en-us/library/ms173156.aspx</a></p>

<p>The slightly longer answer: Maybe some other design pattern would suit you, like the strategy pattern, etc. Inheritance isn't the only way to achieve code reuse.</p>
