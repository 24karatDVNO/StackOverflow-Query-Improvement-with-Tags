<p>If you want to get really awesome with it and have it show up in your Task Manager with an app.exe naming and handle any startup options, you should read into JNI.  JNI will allow you to wrap the starting and stopping of a Java app using a windows executable and it is actually very simple to implement.</p>

<p>If you want something as simple as a windows exe launcher, there are also tools out there such as Launch4j will create exe wrappers for you.</p>
