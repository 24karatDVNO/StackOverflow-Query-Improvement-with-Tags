<p>You'll need to use a <a href="http://msdn.microsoft.com/en-us/library/ms186243%28v=SQL.90%29.aspx" rel="nofollow">Recursive CTE</a> since the values of subsequent columns are dependent upon earlier results.</p>

<p>Do this in pieces, too. Have your first query just return the correct values for <code>Column1</code>. Your next (recursive CTE) query will add the results for <code>Column2</code>, and so on.</p>
