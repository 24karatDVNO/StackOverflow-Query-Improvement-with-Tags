<p>Use a listener class to invoke a method. For e.g....Define a listener in web.xml file. and give a class name in the listener. And now create a servlet class or java class to write a code to invoke the API. </p>

<pre><code>&lt;web-app&gt;
&lt;listener&gt;
    &lt;listener-class&gt;MyServlet&lt;/listener-class&gt;
&lt;/listener&gt;

&lt;/web-app&gt;
</code></pre>
