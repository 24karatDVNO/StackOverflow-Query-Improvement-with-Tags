<p>With the current release of the SDK, SkyDrive functionality is <strong>not available</strong> to third-party developers. As a matter of fact, there is no official public SkyDrive API. There is the unofficial <a href="http://skydriveapiclient.codeplex.com/" rel="nofollow">SkyDrive .NET API Client</a>, but since it is not supported by Microsoft, it will be a bit risky to use it in an app that is outside the "just for fun" zone.</p>
