<p>Select the column/row or cell you want to format:</p>

<pre><code>right button &gt; Format cells &gt; Custom &gt; Type
</code></pre>

<p>There is some examples in the box below, some examples are:</p>

<p>d-mmm-yy -> The single d stands for day of month, without leading zeros
dd-mm-yyyy
yyyy-mm-dd
d/mmm/-yy -> mmm stands for name of the month
dd/mm/yyyy -> You can also specify the separator
yyyy/mm/dd</p>

<p>And so on.</p>
