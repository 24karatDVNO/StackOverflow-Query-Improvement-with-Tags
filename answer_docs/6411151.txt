<p>The open source framework <a href="http://code.google.com/p/impromptu-interface/wiki/UsageMVVM" rel="nofollow">ImpromptuInterface.MVVM</a> uses the C# 4.0 dynamic features to add automatic properties that support property changed. It's ImpromptuViewModel works like an ExpandoObject but also has other features to help with MVVM.</p>
