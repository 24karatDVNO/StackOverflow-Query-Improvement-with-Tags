<p>Visual studio supports a database project type that has folders that can contain "Post-Deployment" and "Pre-Deployment" scripts to run.  MSDeploy may then be used to execute these scripts (as well as other DDL) during the deployment process.  <a href="http://msdn.microsoft.com/en-us/library/aa833410.aspx" rel="nofollow">Here</a> is the documentation on MSDN regarding the use of Pre- and Post-Deployment script definition.</p>
