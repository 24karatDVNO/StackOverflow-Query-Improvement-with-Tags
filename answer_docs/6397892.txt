<p>To communicate with a SQLite database, you can give the following library a try: <a href="https://github.com/ziminji/objective-c-sql-query-builder" rel="nofollow">https://github.com/ziminji/objective-c-sql-query-builder</a></p>

<p>Via this library, you can build SQL queries and construct ORM Models that wil help you better access the database.  The library has a pretty good Wiki page as well that can help you integrate it into your app.</p>
