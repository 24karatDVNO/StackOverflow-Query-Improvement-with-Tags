<p>It should throw an exception because the Server Object Model you are using like <code>SPSite</code>, <code>SPWeb</code> will not work unless it's on the server either web application or windows application.</p>

<p>If you want to get data from a remote sharepoint server you should use SharePoint Web Services.</p>
