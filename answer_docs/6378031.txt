<p><strong><a href="http://en.wikipedia.org/wiki/Class_variable" rel="nofollow">Class variable</a></strong></p>

<blockquote>
  <p>a class variable is a variable defined
  in a class (i.e. a member variable) of
  which a single copy exists, regardless
  of how many instances of the class
  exist</p>
</blockquote>

<p><strong><a href="http://en.wikipedia.org/wiki/Instance_variable" rel="nofollow">Instance variable</a></strong></p>

<blockquote>
  <p>a variable defined in a class (i.e. a
  member variable), for which each
  object of the class has a separate
  copy</p>
</blockquote>

<p>By these definitions, <code>str1</code> is a class variable, and <code>str</code> is not.</p>
