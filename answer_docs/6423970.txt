<p>You can use the <a href="https://stackoverflow.com/questions/6127349/connecting-to-wince-device-from-visual-studio/6127456#6127456">Remote Spy tool</a> to find out the name of your task-bar, then you can try calling <a href="http://msdn.microsoft.com/en-us/library/ms633499%28v=vs.85%29.aspx" rel="nofollow noreferrer">FindWindow</a> function using this name to get the handle, and finally calling <a href="http://msdn.microsoft.com/en-us/library/ms633548%28v=vs.85%29.aspx" rel="nofollow noreferrer">ShowWindow</a> function to get it to show. (In P/Invoke versions for <a href="http://www.pinvoke.net/default.aspx/coredll.FindWindowW" rel="nofollow noreferrer">FindWindow</a> and <a href="http://www.pinvoke.net/default.aspx/user32/showwindow.html" rel="nofollow noreferrer">ShowWindow</a> use <code>coredll.dll</code> instead of <code>user32.dll</code>).</p>
