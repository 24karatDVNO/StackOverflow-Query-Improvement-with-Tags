<p>You'll need to use some sort of API to handle communicating the data from one device to the other.  Each device would send a message to the API, which then knows to transmit that data to some other device.</p>
<p>Using <a href="http://code.google.com/appengine/" rel="nofollow noreferrer">Google Apps Engine</a> and/or <a href="http://code.google.com/android/c2dm/index.html" rel="nofollow noreferrer">C2DM</a> would probably be two good places to start.</p>
