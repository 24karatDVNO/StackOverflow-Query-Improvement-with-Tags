<p>That only prevents multiple inclusions in the same source file; multiple source files <code>#include</code>ing it will still generate multiple definitions of <code>DebugMessage()</code>.  In general, you should either not place functions in header files at all or make them <code>static</code> (and usually <code>inline</code>, since otherwise it doesn't usually make sense to have multiple <code>static</code> definitions of the same function).</p>
