<p>It is loaded by the <code>UIApplicationMain</code> function, which is called from <code>main</code> in the <code>main.m</code> file. The only way to change the name of the nib that is loaded is to change the <code>NSMainNibFile</code> key in your <code>Info.plist</code> file. See the <a href="http://developer.apple.com/library/ios/#documentation/uikit/reference/UIKitFunctionReference/Reference/reference.html#//apple_ref/c/func/UIApplicationMain" rel="nofollow">UIApplicationMain reference</a> for more information.</p>
