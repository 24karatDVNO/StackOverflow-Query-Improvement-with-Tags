<p>Yes, MySQL informs you of the result of each query. Here's what I've just got:</p>

<blockquote>
  <p>mysql> create database foo;</p>
  
  <p>Query OK, 1 row affected (0.01 sec)</p>
  
  <p>mysql> use foo;</p>
  
  <p>Database changed</p>
  
  <p>mysql> show tables;</p>
  
  <p>Empty set (0.00 sec)</p>
  
  <p>mysql></p>
</blockquote>

<p>If you need visual feedback, please consider downloading <a href="http://dev.mysql.com/downloads/workbench/5.2.html" rel="nofollow">MySQL Workbench</a> - it is easy to use and, as a GUI application, is explicit enough to make a new user feel comfortable.</p>
