<p>The command for creating a database is <code>CREATE DATABASE database_name;</code>. Don't forget the semicolon at the end.</p>

<p>If it's successful you should see a message like <code>Query OK, 1 row affected (0.00 sec)</code>. Any query should have at east this much output. If you see no message at all (e.g. you press enter and the next line just looks like a <code>-&gt;</code> prompt) make sure you've closed all of your parentheses and quotation mark and ended the query with a semicolon (<code>;</code>).</p>
