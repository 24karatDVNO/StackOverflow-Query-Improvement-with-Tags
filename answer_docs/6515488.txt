<p>A 405 http response status code means "Method Not Allowed". The Wikipedia article <a href="http://en.wikipedia.org/wiki/List_of_HTTP_status_codes#4xx_Client_Error" rel="nofollow">List of HTTP Status Codes</a> has this definition for the error:</p>

<blockquote>
  <p>A request was made of a resource using
  a request method not supported by that
  resource; for example, using GET on
  a form which requires data to be
  presented via POST, or using PUT on a
  read-only resource.</p>
</blockquote>
