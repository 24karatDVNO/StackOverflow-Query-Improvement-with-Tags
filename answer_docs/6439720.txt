<p>It's incorrect markup to use a block element within an inline one. It may still render if the browser and Doctype are lenient, but it's not valid.</p>

<p>You can usee CSS, however, as discussed <a href="https://stackoverflow.com/questions/746531/is-it-wrong-to-change-a-block-element-to-inline-with-css-if-it-contains-another-b">here</a>.</p>
