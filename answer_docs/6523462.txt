<p>Yes. It is possible. You may have to use the WCF delegation. Delegation is perfect fit for backend service calls with the authentication done on front end.</p>

<p>Please refer to <a href="http://msdn.microsoft.com/en-us/library/ms730088.aspx" rel="nofollow">Delegates and Impersonation</a>. The impersonation level set to Delegate would enable you to authenticate a service running in a different machine from the caller server.</p>
