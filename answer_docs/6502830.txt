When creating cache you can define policy which contains ChangeMonitors property.I think adding SqlCacheDepency item to  ChangeMonitors property is what you looking for. Take a look a this classes <a href="http://msdn.microsoft.com/en-us/library/system.web.caching.sqlcachedependency.aspx" rel="nofollow">SqlCacheDepency</a>
 , <a href="http://msdn.microsoft.com/en-us/library/system.runtime.caching.cacheitempolicy.aspx" rel="nofollow">CacheItemPolicy</a>. I can provide code if you need it.</p>
