<p>You can make your app use the entire iPad screen and not run in the simulator simply by changing the target device from iPhone to iPhone/iPad in Xcode. If you want your app to show different UI and run different code on the iPad, you can check for the device by using one of <a href="https://stackoverflow.com/q/4567728/224988">several ways to detect an iPad</a> in Objective-C.</p>
