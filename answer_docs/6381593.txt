<p><code>&lt;&lt;&lt;</code> is the opener for the HEREDOC syntax. You follow it by an arbitrary string which is used as terminator. It's alternative syntax to quotes. See: <a href="http://www.php.net/manual/en/language.types.string.php#language.types.string.syntax.heredoc" rel="nofollow">http://www.php.net/manual/en/language.types.string.php#language.types.string.syntax.heredoc</a></p>
