<p>Check the visibility of the class with which content assist does not work. If you are trying to call a method of that class, then check the visibility of that method as well. If you want that class/method to be accessible outside the package the you have to make that class/method <code>public</code> You can learn more about the visibility modifiers <a href="http://download.oracle.com/javase/tutorial/java/javaOO/accesscontrol.html" rel="nofollow">here</a></p>
