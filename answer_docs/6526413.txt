<p>Put your inline styles in a document and name it with the extension .css. Then reference it between your  tags with the following line.</p>

<pre><code>&lt;link rel="stylesheet" href="path/to/file.css" type="text/css" /&gt;
</code></pre>

<p>The href can be both relative (../images/file.css) or absolute (/images/file.css).</p>

<p>If you need to access different media, you can put in a media tag (like media="screen") or media="print" for stylesheet only to be used when printing a page.</p>
