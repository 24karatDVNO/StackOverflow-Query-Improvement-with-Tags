<p>You could allocate an unmanaged array and use <a href="http://msdn.microsoft.com/en-us/library/ms146625.aspx" rel="nofollow"><code>Marshal.Copy</code></a> to copy the contents of the managed <code>byte[]</code> array to an unmanaged <code>unsigned char*</code> array. Note that unmanaged resources must be cleaned up manually.</p>
