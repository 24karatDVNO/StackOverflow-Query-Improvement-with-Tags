<p>The HStringTemplate package provides a very general runtime templating system for not only HTML, but any sort of text output: </p>

<p><a href="http://hackage.haskell.org/packages/archive/HStringTemplate/0.6.6/doc/html/Text-StringTemplate.html" rel="nofollow">http://hackage.haskell.org/packages/archive/HStringTemplate/0.6.6/doc/html/Text-StringTemplate.html</a></p>

<p>The HStringTemplate package integrates with Happstack nicely through a few extra instances provided by an optional package: <a href="http://hackage.haskell.org/package/happstack-hstringtemplate" rel="nofollow">http://hackage.haskell.org/package/happstack-hstringtemplate</a></p>
