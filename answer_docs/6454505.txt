<p>You have to make the browser "visit" the page for the download dialog to appear.  Just sending a post request is not enough.</p>

<p>Probably the best way to do this with JavaScript is to dynamically make a <code>&lt;form&gt;</code> object with <code>action="/your/servlet/page"</code> and then use JavaScript to call <code>submit()</code> on the form object (don't forget to add the <code>&lt;form&gt;</code> to the page or it won't work).</p>
