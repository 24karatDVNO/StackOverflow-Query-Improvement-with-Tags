<p>If intent was sent via <code>startActivity</code>, <code>startActivities</code> or <code>startActivityForResult</code> an <code>ActivityNotFound</code> exception is thrown.</p>

<p>If intent was sent via <code>startService</code> then nothing bad happens, and <code>startService</code> just returns <code>null</code>. And if <code>bindService</code> is used, function returns <code>false</code>.</p>
