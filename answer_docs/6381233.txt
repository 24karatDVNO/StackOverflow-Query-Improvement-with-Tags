<p>Most browsers can do this automatically with <a href="http://diveintohtml5.info/forms.html" rel="nofollow">HTML5 validation</a>. You won't have much control over how it looks but it's 1000x easier to code and works without Javascript.</p>

<p>If you want more visual control there's <a href="http://flowplayer.org/tools/validator/index.html" rel="nofollow">jQuery Tools Validator</a>. Although this uses Javascript it should fall back to HTML5 if Javascript is disabled.</p>
