<p>SASL supports the notion of using a username to authenticate. Your directory server administrator may be able to configure the directory server to map distinguished names to identities. Given the correct mapping it is possible for a client to authenticate without knowing the distinguished name. Professional-quality directory servers support a number of different mapping mechanisms such as <code>direct mapping</code>, <code>exact match</code>, <code>regular expression</code>, or a custom identity mapper. </p>
