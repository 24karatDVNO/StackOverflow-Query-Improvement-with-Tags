<p>The Spring Framework ships with a number of <a href="http://static.springsource.org/spring/docs/3.0.x/reference/remoting.html" rel="nofollow">remoting options</a> that are all very easy to setup. You can use their classes for simpler configuration of something standard like RMI or JMS, or use a lightweight web services protocol such as Spring's HTTP invoker or Hessian.</p>
