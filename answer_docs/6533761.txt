<p>Use the PRG pattern, which can be described as:</p>

<pre><code>Never show pages in response to POST
Always load pages using GET
Navigate from POST to GET using REDIRECT
</code></pre>

<p>See <a href="http://en.wikipedia.org/wiki/Post/Redirect/Get" rel="nofollow">http://en.wikipedia.org/wiki/Post/Redirect/Get</a> and links from there</p>
