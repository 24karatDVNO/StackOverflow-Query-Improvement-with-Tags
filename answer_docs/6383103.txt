<p>You can add the subtitle just the same way (index/node/pages/subtitle/bar) but for the text property you might want a fulltext index. Documentation on indexing is <a href="http://docs.neo4j.org/chunked/snapshot/rest-api-indexes.html" rel="nofollow">here</a> and how to create a fulltext index is <a href="http://docs.neo4j.org/chunked/snapshot/rest-api-indexes.html#rest-api-create-node-index-with-configuration" rel="nofollow">there</a>. At some point <a href="http://docs.neo4j.org/chunked/snapshot/rest-api-batch-ops.html" rel="nofollow">batch operations</a> should make sense, but this feature is considered experimental so far.</p>
