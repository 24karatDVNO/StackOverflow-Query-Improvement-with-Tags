<p><a href="http://www.webmproject.org/" rel="nofollow">WebM</a> , is a Video codec + file format created created exactly for streaming video over web.It is free(BSD license) code + documentation + standard.</p>

<p>Low computational footprint to enable playback on any device, including low-power netbooks, handhelds, tablets, etc.*</p>

<p>Simple container format</p>

<p>Highest quality real-time video delivery</p>

<p>Click and encode. Minimal codec profiles, sub-options; when possible, let the encoder make the tough choices.</p>
