<p>It probably is the browser scrollbar, but you might also want to ensure that the presentation displays in the mode you expect by setting the ShowType, which can take on three values: </p>

<p>ppShowTypeSpeaker
ppShowTypeKiosk
ppShowTypeWindow</p>

<p>Only the window type will display a scrollbar or any other "furniture".  The other types go full screen, but it would probably be best to specify which you want rather than depending on it being set that way already.</p>
