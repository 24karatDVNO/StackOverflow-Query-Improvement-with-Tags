<p>GET parameters are just the query strings appended to the end of the URL.  Thus, to set the parameters <code>foo</code> and <code>bar</code> to <code>42</code> and <code>cubiculum</code>, respectively, the URL given in the <code>IMG</code>'s <code>SRC</code> attribute would end with <code>?foo=42&amp;bar=cubiculum</code>.</p>
