<p>You can't extend <code>Double</code> class, since it is final.</p>

<p>But you can extend <code>Number</code> class and implement <code>Comparable</code> interface - the way actual <a href="http://download.oracle.com/javase/6/docs/api/java/lang/Double.html" rel="nofollow">Double class is created</a>.</p>
