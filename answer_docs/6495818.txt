<p>If you're asking about posting the data to a web server, I'd recommend using <a href="http://allseeing-i.com/ASIHTTPRequest/" rel="nofollow">ASIHTTPRequest</a> to HTTP POST the data to the server in Json format. You'd then use the PHP function <a href="http://php.net/manual/en/function.json-decode.php" rel="nofollow">json_decode()</a> to decode the data. </p>

<p><a href="https://github.com/stig/json-framework/" rel="nofollow">Json-framework</a> is a good Json framework for the iPhone. </p>
