<p>You should also check out <a href="http://scrapy.org" rel="nofollow">Scrapy</a>, a django app:</p>

<blockquote>
  <p>Scrapy is a fast high-level screen scraping and web crawling framework, used to crawl websites and extract structured data from their pages. It can be used for a wide range of purposes, from data mining to monitoring and automated testing.</p>
</blockquote>

<p>It crawls the sites and extracts the data of interest, which you can specify using xpath across the site periodically, and saves it to the database as a new version.</p>
