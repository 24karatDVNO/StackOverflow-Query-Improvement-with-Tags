<p>You could use a combination of templates (tmpl) and $.ajax.  After the new appointment has been added to the database you can call a JavaScript function to invoke an $.ajax call.  In the callback function you can then inject the newly updated json data into a template.  This type of workflow will allow you to update lists without a page refresh.</p>
