<p>Aside from exploiting some unpatched security vulnerability, if you're running on a Windows OS and using VBScript, you would need to be able to authenticate to \kullanici with an account that has access to the d$ share. Normally, only administrators would have access to a hidden administrative share like that.</p>

<p>If \kullanici is a Windows 2000 computer though, if you gave Everyone access to the d$ share and to the D:\lotus\notes\data\archive folder, then any anonymous user would have access to it.</p>
