<p>The main idea is to produce a HTTP 301 response code, which correspond to a permanent redirect.</p>

<p>Search engines like this, because they know the page is no longer here, but at the new address. It's also transparent for the user.</p>

<p>The challenge is sometimes to be able to map the old url to the new one... but it's the more clean way to move pages...</p>
