<p>jQuery 1.4.1 have native support for parsing JSON at: <a href="http://api.jquery.com/jQuery.parseJSON/" rel="nofollow">http://api.jquery.com/jQuery.parseJSON/</a>.</p>

<blockquote>
  <p>Additionally if you pass in nothing, an empty string, null, or
  undefined, 'null' will be returned
  from parseJSON. Where the browser
  provides a native implementation of
  JSON.parse, jQuery uses it to parse
  the string.</p>
</blockquote>
