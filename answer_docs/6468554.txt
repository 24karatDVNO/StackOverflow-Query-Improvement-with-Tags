<p>You can add <strong>any information</strong> on expanding of the subgrid. You should just implement the corresponding behavior in your <a href="http://www.trirand.com/jqgridwiki/doku.php?id=wiki:subgrid#events" rel="nofollow noreferrer">subGridRowExpanded</a> event handler. See <a href="http://www.ok-soft-gmbh.com/jqGrid/SubGrid.htm" rel="nofollow noreferrer">the demo</a> as an example from <a href="https://stackoverflow.com/questions/4114498/jqgrid-how-can-we-make-an-custom-row-detail/4115481#4115481">the answer</a>.</p>
