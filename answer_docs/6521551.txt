<p><a href="http://bama.ua.edu/cgi-bin/man-cgi?comm" rel="nofollow"><code>comm</code></a> is a useful tool for comparing files.</p>

<blockquote>
  <p>The comm utility will read <em>file1</em> and <em>file2</em>, which should  be
       ordered in the current collating sequence, and produce three
       text columns as output: lines only in <em>file1</em>; lines  only  in
       <em>file2</em>; and lines in both files.</p>
</blockquote>
