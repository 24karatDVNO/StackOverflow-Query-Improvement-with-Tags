<p>Yes, the tags app should be able to recognize v-card data.</p>

<p>I've used the NXP TagWriter app to write a vcard to a Mifare Classic card, then read it in via the Tags app.  It showed up as a contact type and allowed me to import the contact into the contacts system.</p>

<p>I've also had success in using the Tags app to read vcard data off of a demo NFC-enabled business card.</p>
