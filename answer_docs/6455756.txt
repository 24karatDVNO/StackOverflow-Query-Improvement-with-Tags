<p>There are certain patterns and tools that you can use from project to project that will assist in the conversion of a Photoshop document into a standards compliant site.</p>

<p>The best advice is to learn CSS like the back of your hand. This will allow you to slice psd files correctly - only using images that are necessary. It will also help you understand the limits you must work within, in Photoshop or any other image tool, to create websites. The stronger your CSS knowledge becomes the higher the limit...</p>
