<p>I believe you will have to insert the loaded HTML into the DOM before the browser will start loading any of the images. You should be able to insert it into a hidden (<code>display:none</code>) div, and set the <code>.load()</code> callback on that hidden div.</p>

<p><a href="https://github.com/peol/jquery.imgloaded/raw/master/ahpi.imgload.js" rel="nofollow">This plugin</a> improves the <code>.load()</code> event so that it is more consistently triggered on image loading.</p>
