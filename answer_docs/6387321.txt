<p>The typical way to center an element using CSS is with <code>margin-left:auto; margin-right:auto;</code>.</p>

<p>In addition, the element needs to be displayed as a block and have a fixed width.</p>

<p>So your CSS would look something like this:</p>

<pre><code>.myelement {
    margin-left:auto;
    margin-right:auto;
    display:block;
    width:100px;
}
</code></pre>

<p>Hope that helps.</p>
