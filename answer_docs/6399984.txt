<p>Facebook uses Bigpipe, there is an open implementation in the works called openpipe</p>

<p>Bigpipe pushes the content to the browser when server stopped processing, so user will notice that it is faster.</p>

<p>It basically loads pagelets, when they are ready for the user, at the browser the implementation is Javascript based, and you must push the info to the client with your preferred server language.</p>
