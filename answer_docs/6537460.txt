<p>You can exclude the anchor selection by using <code>div:target</code></p>

<pre><code>  * {
        color: black;
    }
    div:target{
        color: red;
    }
</code></pre>

<p><a href="http://jsfiddle.net/" rel="nofollow">http://jsfiddle.net/</a></p>

<p>With the same technique you can try with <code>a:target</code> to get only the current targeted anchor</p>
