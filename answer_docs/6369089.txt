<p>Encoding/decoding JSON in Django is not going to be an issue performance-wise, although I'd point out that <a href="https://github.com/simplejson/simplejson" rel="nofollow"><code>simplejson</code></a> is a bit more performant than the <code>json</code> module in Python 2.6/2.7, so you might want to use that.</p>
