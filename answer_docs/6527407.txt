<p>You can include a 3-state switch that: </p>

<p>) does data entry and re-displays the cleared form;</p>

<p>) displays the error and re-populates the form so user can fix it;</p>

<p>) default to no data entry, clear form</p>

<p>Use a post or session variable to specify the action to take, default to no action if the variable is not set or does not validate.</p>
