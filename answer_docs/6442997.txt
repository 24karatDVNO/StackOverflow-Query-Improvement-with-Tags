<p>Some scripting languages have tools that can help you. For example <code>use strict</code> in perl requires declaration of each variable before using. But still the language is weakly typed by definition. </p>

<p>Sometimes naming conventions help. For example we inherited from good old Fortran tradition that int variables' names should start from i, j, k, l, m, n. And this convention is used now at least for indexes. </p>
