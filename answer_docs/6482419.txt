<p>Any AD group type is usable by SharePoint so long as that group is usable by the server SharePoint is running on.  Said another way, if you were using the OS level tools on the server and the OS recognizes your group, then you can use it in SharePoint.  </p>

<p>As for when group memberships changes become effective, it has always been near real time for me but I can't say that I can speak to all possible AD topology deployments.</p>
