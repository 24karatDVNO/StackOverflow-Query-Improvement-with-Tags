<p>In the <code>Runnable</code> of your <code>MenuItem</code> just have it push a new screen that either has a constructor that you feed the Strings into, or expose some methods on your screen that set the Strings. </p>

<p>If you don't know how to parse JSON (and are using an OS other than 6), check out org.json.me. If you're using 6, the JSON classes are already included.</p>
