<p>If you need to share controls between web sites you need to implement custom server control.</p>

<p><a href="http://msdn.microsoft.com/en-us/library/zt27tfhy.aspx" rel="nofollow">Walkthrough: Developing and Using a Custom Web Server Control</a></p>

<p>User controls (ascx) are not supposed to be shared between web sites.</p>

<p>Although you can reference the same physical file from different projects, this is a bad approach.</p>
