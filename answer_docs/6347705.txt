<p><code>$</code> is an object used in the <a href="http://jquery.com/" rel="nofollow">jQuery</a> library. You need to include the jQuery library in your code to use it.</p>

<p>Include jQuery by adding the following line after the <code>&lt;head&gt;</code> tag in your HTML (before you use code that relies on jQuery):</p>

<pre><code>&lt;script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.1/jquery.min.js"&gt;&lt;/script&gt;
</code></pre>
