<p>In addition to the maxmind system, there's also <a href="http://ipinfodb.com/ip_database.php" rel="nofollow">ipinfodb</a>. However IME, I wouldn't expect an average accuracy of better than 300 miles.  </p>

<p>HTML5 provides a <a href="http://dev.w3.org/geo/api/spec-source.html" rel="nofollow">javascript API</a> to determine location on the client by various methods. Where available, this is much more accurate - but it does result in a prompt to the user to permit capture of the data - and if you've no GPS / Wifi connectivity, it will just timeout.</p>
