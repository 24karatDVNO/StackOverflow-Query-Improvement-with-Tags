<pre><code>public static double longBitsToDouble(long bits)
</code></pre>

<blockquote>
  <p>Returns the double value corresponding
  to a given bit representation. The
  argument is considered to be a
  representation of a floating-point
  value according to the IEEE 754
  floating-point "double format" bit
  layout.  </p>
</blockquote>

<p><strong>Parameters:</strong>  <code>bits - any long integer.</code></p>

<p><strong>Returns:</strong> <code>the double floating-point value with the same bit pattern.</code></p>
