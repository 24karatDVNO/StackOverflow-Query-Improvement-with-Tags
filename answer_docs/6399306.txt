<p>XML or Arrays as the data type that feeds a grid is really a property of the grid's store.  That data is given to the grid as records of the store, so its just the store's reader that cares about the incoming data type.  Therefore, you should be able to use this example to put XML data in a store, which can be used by a grid:</p>

<p><a href="http://dev.sencha.com/deploy/ext-4.0.2a/examples/grid/xml-grid.html" rel="nofollow">http://dev.sencha.com/deploy/ext-4.0.2a/examples/grid/xml-grid.html</a> </p>
