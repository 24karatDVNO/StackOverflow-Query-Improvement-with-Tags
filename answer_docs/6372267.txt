<p>The only way you can programmatically manipulate the displayed page of a panorama control is to add and remove from the collection of items. This causes no end of complications and is in any case not a supported operation.</p>

<p>You <em>can</em> set the SelectedItem of a Pivot control like this.</p>

<p>However, as several people have already commented, the style guide specifically and unambiguously tells you not to use the application bar in Panoramic or Pivot applications.</p>
