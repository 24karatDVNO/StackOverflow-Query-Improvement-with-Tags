<p>If you have a network graph of dependencies and you want them to scale, a graph database is the best solution. <a href="http://neo4j.org/" rel="nofollow">Neo4J</a> is a popular, powerful database for tracking this type of dependencies.</p>

<p>There are several ways to interface with Neo4J from Ruby:</p>

<ul>
<li>You can use JRuby and its java interface.</li>
<li>Use its REST API</li>
<li>Use neo4j.rb or one of the other <a href="http://wiki.neo4j.org/content/Ruby" rel="nofollow">Ruby interface libraries</a>.</li>
</ul>
