<p>You have two functions there. The inner function returns a value, but not the outer one.  The inner function is an event handler so the return value doesn't go anywhere.  Your XMLHttpRequest is asynchronous, so you won't get a return value right away.  See this post for a more detailed explanation: <a href="https://stackoverflow.com/questions/6461958/parameter-true-in-xmlhttprequest-open-method/6462145#6462145">parameter &quot;true&quot; in xmlHttpRequest .open() method</a></p>
