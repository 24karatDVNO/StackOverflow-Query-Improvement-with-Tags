<p>SHA-512 is a one-way hash, meaning you can't take the hash value and determine what string/value was used to generate that hash. Technically, there can be an infinite number of strings/values that would generate the same hash.</p>

<p>If you want to be able to get the value back, you should use <a href="http://www.codeproject.com/KB/security/SimpleEncryption.aspx" rel="nofollow">encryption</a>, not hashing.</p>
