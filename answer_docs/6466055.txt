<p>Using jQuery's <a href="http://api.jquery.com/click/" rel="nofollow">.click()</a> function you can attach a function to run whenever the selected element is clicked by the user.</p>

<p>You will also probably want to use <a href="http://api.jquery.com/jQuery.getJSON/" rel="nofollow">$.getJSON()</a> instead of manually attaching the <code>&lt;script&gt;</code> tag.</p>
