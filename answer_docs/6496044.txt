<p>$have_reconsidered{lc($name)} is the value of the hash %have_reconsidered that belongs to the key lc($name).  This value could definitely be numeric, but even if it is a string, it can still be auto-incremented (see <a href="http://perldoc.perl.org/perlop.html#Auto-increment-and-Auto-decrement" rel="nofollow">http://perldoc.perl.org/perlop.html#Auto-increment-and-Auto-decrement</a>).</p>
