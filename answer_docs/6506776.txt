<p>AFAIK, <a href="http://www.h2database.com/html/features.html#embedded_databases" rel="nofollow">H2</a> and <a href="http://hsqldb.org/doc/guide/ch01.html#N101A8" rel="nofollow">HSQLDB</a> provide only file-based and in-memory connection URIs when used in "standalone" mode. So a file, or some heap memory space, shall be needed; but you don't give it a File/InputStream, everything is in the connection URI, eg <code>jdbc:h2:~/test</code>.</p>
