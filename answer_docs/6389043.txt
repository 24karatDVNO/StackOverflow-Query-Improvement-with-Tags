<p>There are two things that you need to learn.</p>

<ol>
<li>Tools</li>
<li>APIs</li>
</ol>

<p>for tools look at: <a href="http://www.gnu.org/software/make/manual/" rel="nofollow">make</a> and  <a href="http://gcc.gnu.org/onlinedocs/gcc-4.4.6/gcc/" rel="nofollow">GCC</a>. Make forms the basis most unix builds although there are a number of tools (Auto tools and CMake) that will generate the Makefile for you.</p>

<p>For APIs the best doc is the <a href="http://www.gnu.org/s/libc/manual/" rel="nofollow">GLibC Manual</a>. It does a great job of explaining the basic unix apis.</p>
