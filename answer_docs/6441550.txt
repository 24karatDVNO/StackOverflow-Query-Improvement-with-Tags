<p>The parameter of the <code>.load()</code> method must be some representation of HTML code, whether it be a div, or an entire page.  It seems as if <code>'/return_result'</code> doesn't meet those requirements. If I am wrong, and the element does contain HTML, try using the jQuery <code>$('#div').empty()</code> and <code>$('#div').appendTo('destination')</code> methods.</p>
