<p>You have to set the Canvas.Top and Canvas.Left attached properties.</p>

<p>For example:</p>

<pre><code>Canvas.SetTop(path, 10);
Canvas.SetLeft(path, 10);
</code></pre>

<p>More information about attached properties can be found here: <a href="http://msdn.microsoft.com/en-us/library/ms749011.aspx" rel="nofollow">Attached Properties MSDN</a></p>
