<p>A general answer for a general question:</p>

<ul>
<li>Use the Designer to create your windows; hide and show the auxiliary windows as needed.</li>
<li>Use a flow manager class to manage the visibility of a related set of windows.</li>
<li>The stacked widget is useful for managing a button/icon whose appearance changes based on state; the different representations live in the stack.</li>
</ul>
