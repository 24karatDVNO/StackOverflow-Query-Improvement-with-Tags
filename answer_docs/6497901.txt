<p>Sharing should be done with the ACTION_SEND Intent: <a href="http://developer.android.com/reference/android/content/Intent.html#ACTION_SEND" rel="nofollow">http://developer.android.com/reference/android/content/Intent.html#ACTION_SEND</a></p>

<p>For example if you use this with a MIME type of image/*, the user will be prompted to select all of the things they can share an image with.  If Picassa is one of these on their device, it will be in the list.</p>
